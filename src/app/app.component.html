NPM Shopping-Cart DashBoard

NPM Module

This is a simple module to add items/products to the shopping cart

Use the below command to install the npm package in your node application

npm i shopping-cart-example --save

'shopping-cart-example' module has 2 functions

  1. addToCart(...itemObject) - To add an object containing the product details to the cart
  2. allItemsInCart() - To display all the items in the cart

Example:

addToCart(... itemObject) - Takes a itemObject as a REST parameter


const addingToCart = require('shopping-cart-sample')

productName: "Sunglasses",
